| 冠词类型     | 用法场景              | 例子                        |
| -------- | ----------------- | ------------------------- |
| **the**  | 特指某个已知的事物         | **The car is expensive.** |
| **a/an** | 泛指某个不确定的事物        | **I need a car.**         |
| **零冠词**  | 泛指复数名词、不可数名词或专有名词 | **Cats are cute.**        |
