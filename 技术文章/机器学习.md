# 3、归纳偏好

机器学习算法在学习过程中对某种类型假设的偏好，任何一个有效的机器学习算法必有其偏好
一般原则：奥卡姆剃刀 -> 若非必要，勿增实体

# 4、NFL定理

NFL定理：一个算法若在某些问题上a比另一个算法b好，必然存在另一些问题，b比a好

# 5、泛化误差和经验误差

泛化误差：在未来样本上的误差
经验误差：在训练集上的误差，也叫训练误差

经验误差不是越小越好，可能会出现过拟合和欠拟合
